# Smart Medicine Reminder

## Problem Statement
Many people forget to take medicines on time, which can affect their health and recovery.

## Solution
Smart Medicine Reminder helps users:
- Add medicines
- Store dosage and timings
- View all medicines in a dashboard
- Get reminder alerts
- Access medicine information through an AI assistant

## Features
- Add Medicine
- View Medicines
- Delete All Medicines
- Reminder Check
- AI Medicine Assistant

## Tech Stack
- Python
- Streamlit
- Pandas

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
